# Assets Directory

This directory contains images and videos for the README.md file.

## Required Files

Place the following files in this directory:

### 1. Title Image
- **File**: `title-image.png` (or `.jpg`)
- **Description**: Title/banner image for the library
- **Recommended size**: 1200x400 pixels (or similar aspect ratio)
- **Usage**: Displayed at the top of README after badges

### 2. Hardware Image
- **File**: `hardware-image.png` (or `.jpg`)
- **Description**: Photo or diagram of the hardware setup
- **Recommended size**: 800x600 pixels (or similar aspect ratio)
- **Usage**: Displayed in the Hardware Requirements section

### 3. Demo Video/GIF
- **File**: `advanced-example-demo.gif` (or `.mp4`, `.webm`)
- **Description**: Animated demo showing the AdvancedExample in action
- **Recommended format**: GIF (auto-plays on GitHub) or video
- **Recommended size**: 640x480 pixels or larger
- **Recommended duration**: 10-30 seconds
- **Usage**: Displayed in the AdvancedExample Walkthrough section

## File Formats

- **Images**: PNG (preferred) or JPG
- **Video/GIF**: GIF (preferred for GitHub) or MP4/WebM

## Notes

- All assets should be optimized for web (reasonable file sizes)
- GIFs should be optimized to reduce file size
- Images should be high quality but compressed appropriately
